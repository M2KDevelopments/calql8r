## Using Single Letter Values to Represent Functions
### Trigonometry
<ul>
    <li>Sin = S</li>
    <li>Sinh = s</li>
    <li>Cos = C</li>
    <li>Cosh = c</li>
    <li>Tan = T</li>
    <li>Tanh = t</li>
</ul>

### Logarithmns and Exponentials
<ul>
    <li>e = e</li>
    <li>ln = E</li>
    <li>log10 = L</li>
    <li>alogb = l</li>
</ul>

### Constants
<ul>
    <li>PI = p</il>
<ul>


### Others Functions
<ul>
    <li>Square Root = 2r</li>
    <li>Cube Root = 3r</li>
    <li>Root = r</li>
    <li>Pol = P</li>
    <li>Rec = R</li>
<ul>
